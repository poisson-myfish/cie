Make your first app!

This guide is meant for people that never typed code in their life but want to learn to make an app really fast and easy. If you're one of them, let's get started!

Set up CIE

Step 1: Download CIE and get used to the Command Line

To use CIE and make apps, you first have to download it and learn how to use the command line. The command line is a very important part of developing apps, as it is used by most programming languages to run and manage your app. The command line is really easy to learn and it will help you a lot in your programming journey.

  • Go to the python website and download python.
  • Open the file you just downloaded. While installing, make sure you check the 'Add to PATH' check box.
  • Go to the CIE Releases Page and download the latest ZIP containing CIE.
  • Unzip the files in a good location, like Documents/CIE
  • If you're on Mac or Linux, open the "Terminal". If you're on Windows, open the "Command Prompt". If you haven't seen a Console Window before, don't fear, it's easier than it looks like.
  • Learn to use the Command Line basic commands.
    • First, check the folder you're currently in. This is shown on the last line of your window
    • On Mac and Linux, use ls and on Windows dir to show the files in the current directory.
    • Use cd to navigate to a directory. For example, if you're in Documents, type cd CIE to navigate to the CIE directory in Documents.
    • Use cd .. to go a folder back

Step 2: Make a Hello, World app and run it

A Hello, World app is a simple program which, as the name suggests, prints "Hello, World!" to the screen. It is often used to check if you have everything set up correctly, so let's do that!

  • Now that you know how to use the command line, navigate to the directory in which you extraced CIE.
  • Open a text editor like Notepad and type the following lines into a document: say "Hello, World!". IMPORTANT: Make sure you press enter after the line you wrote, so that you have a blank space after it.
  • Save the file in the directory you just extracted CIE with the filename hello.cie. Make sure that you don't have a .txt extension at the end.
  • Go back to the command line and type: python main.py hello.cie. You should see Hello, World! print to the screen.

Learn basic programming

Part 1: Variables and user input

Variables are used to store data under a specific name. For example, if you use the sentence "My Company is awesome" a lot throughout your program, you can store it in a variable and access it under the name of "company_description". This way, if you ever have to edit this description, you will only do it in one place. But enough theory, let's get to programming!

Delete everything from the program you made earlier and add the following line: set name to "Bob". So, what does that mean? By using the set keyword, you created your first variable! It's name is, well, "name". And it's value is "Bob".

Now, let's use the variable that we just created! On the next line of your program, type: say get name. What does this mean? It means that you print(say) the variable name to the screen. The get keyword is used to get the variable. You have to type get before using any variable.

Now your program should look like this:

set name to "Bob"
say get name

And again, it is very important to leave a blank line after you finish your program, showing where it ends. If you don't do that, the last line of your program will not run.

Now let's make this more exciting! Replace set name to "Bob" with set name to user-input. This means that instead of giving the variable a value, you request it from the user. So now if you run the program, it will, apparently, not do anything. But you will see a blinking cursor, which means it's waiting for input. So if you type your name and press enter, it will print your name to the screen.

After you did all this, you hopefully learned how to use variables and print them to the screen. Now, try to delete everything from the program and make it again, to see if you remember everything.
